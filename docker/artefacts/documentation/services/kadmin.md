# Kadmin

Web Based Kafka Consumer and Producer.

**[Documentation](https://github.com/BetterCloud/kadmin)** | **[GitHub](https://github.com/BetterCloud/kadmin)**

### How to enable?

```
platys init --enable-services KADMIN
platys gen
```

### How to use it?

Navigate to <http://192.168.1.112:28106>
